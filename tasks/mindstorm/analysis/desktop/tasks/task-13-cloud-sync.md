# Task 13: Cloud 同步与多设备

## Feature
实现本地优先的数据同步: 可选云备份、跨设备设置同步。

## 优先级
P3 - 锦上添花

## 验收标准
- [ ] 本地优先架构: 所有数据 SQLite 本地存储
- [ ] 可选云同步: Turso Embedded Replicas 或自建 API
- [ ] 设置同步: 用户偏好跨设备同步
- [ ] 工作空间同步: 工作空间配置云备份
- [ ] 离线模式: 完全离线可用
- [ ] 冲突解决: 简单的最后写入胜出策略
- [ ] 同步状态指示器
- [ ] 同步频率配置

## 推荐方案

### 个人用户 (≤3人)
纯本地，无需云同步

### 小团队 (3-10人)
```
SQLite (本地) ←→ Turso (云端)
                  ↑↓
              其他设备 SQLite
```

- Holaboss: 工作空间可移植性

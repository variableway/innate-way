# Crazy Thoughts

## 问题

1. 如果IDE，Terminal这种是开发人员的必备工具，那么对于普通人来说他们的必备工具是什么？
2. 如果普通人的也有类似一个IDE这样的工具，但是没有那么复杂，就是做什么什么事情呢：
   1. 获取最新感兴趣的新闻，或者自己关注的新闻和社交媒体信息- Feeds
   2. 可以在工具中使用AI完成自己想要的任务，这个工具是触发AI的起点
   3. 可以收集自己的IDEA，然后保存，然后筛选，这个过程AI可以帮忙执行
   4. 想要做的就用AI进行coding，这个工具是起点发起调用AI
   5. 进行的中间结果和实际结果可以检查到
3. 达到一种理想的生活方式就是:
   1. 不焦虑有序，任务进来分析，决定做不做，做就是AI执行了
   2. 可以一早起来查看任务，然后使用工具安排好之后，就可以用AI来执行所有的事情
   3. 这个执行到完成的过程可以看到，可以从手机上访问Bot直接让他停止，或者吩咐说明
   4. 所有过程中的文件都需要记录，形成自己的LLM，可以参考LLM-Wiki
   5. 人可以放松一点，做什么事情的时候就专心做什么，可以不担心，因为AI可以24小时运行
